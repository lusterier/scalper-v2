## Goal

Add `HistoricalOHLCSource` async-iterable class in `packages/exchange/paper/historical_ohlc_source.py` that streams `OHLCRow` instances from the `ohlc_1m` TimescaleDB hypertable with **configurable pace** (`1x` real-time / `10x` accelerated / `max` no-wait) per BRIEF §12.2:1955. ## Algorithm specification

```python
Pace = Literal["1x", "10x", "max"]

_PACE_FACTOR: dict[str, float] = {"1x": 1.0, "10x": 10.0, "max": float("inf")}

@dataclass(frozen=True, slots=True)
class OHLCRow:
    symbol: str
    bucket_start: datetime
    open: Decimal
    high: Decimal
    low: Decimal
    close: Decimal
    volume: Decimal
    source: str  # 'binance' | 'bybit'


class HistoricalOHLCSource:
    """Async-iterable cursor-streamed historical OHLC replay with pace control.

    See module docstring for full algorithm + L-011-N/A note + cursor+tx
    lifetime caveat + boundary-check belt-and-suspenders rationale.
    """

    def __init__(
        self,
        pool: asyncpg.Pool,
        *,
        symbols: list[str],
        from_at: datetime,
        to_at: datetime,
        pace: Pace = "max",
        source: str = "binance",
        now_fn: Callable[[], datetime] = now_utc,
    ) -> None:
        if not symbols:
            raise ValueError("HistoricalOHLCSource: symbols must not be empty")
        if to_at <= from_at:
            raise ValueError("HistoricalOHLCSource: to_at must be > from_at")
        if pace not in _PACE_FACTOR:
            raise ValueError(f"HistoricalOHLCSource: invalid pace {pace!r}")
        if source not in ("binance", "bybit"):
            raise ValueError(f"HistoricalOHLCSource: invalid source {source!r}")
        self._pool = pool
        self._symbols = list(symbols)
        self._from_at = from_at
        self._to_at = to_at
        self._pace_factor = _PACE_FACTOR[pace]
        self._source = source
        self._now_fn = now_fn

    async def __aiter__(self) -> AsyncIterator[OHLCRow]:
        async with self._pool.acquire() as conn, conn.transaction():
            replay_start_real: datetime | None = None
            first_bucket_at: datetime | None = None
            async for row in conn.cursor(
                _SELECT_OHLC_REPLAY_SQL,
                self._symbols,
                self._from_at,
                self._to_at,
                self._source,
                prefetch=_DEFAULT_PREFETCH,
            ):
                ohlc = _row_to_ohlc(row)
                if replay_start_real is None:
                    replay_start_real = self._now_fn()
                    first_bucket_at = ohlc.bucket_start
                elif self._pace_factor != float("inf"):
                    # Compute target real-time elapsed since replay start.
                    sim_dt = (ohlc.bucket_start - first_bucket_at).total_seconds()
                    target_real_dt = sim_dt / self._pace_factor
                    elapsed = (self._now_fn() - replay_start_real).total_seconds()
                    sleep_for = max(0.0, target_real_dt - elapsed)
                    if sleep_for > 0:
                        await asyncio.sleep(sleep_for)
                yield ohlc
```

`_SELECT_OHLC_REPLAY_SQL`:
```sql
SELECT symbol, bucket_start, open, high, low, close, volume, source
FROM ohlc_1m
WHERE symbol = ANY($1)
  AND bucket_start >= $2
  AND bucket_start < $3
  AND source = $4
ORDER BY bucket_start ASC
```

`_DEFAULT_PREFETCH = 1000` (mirror T-504; §N9-exempt infra const per L-001).

`_row_to_ohlc(row) -> OHLCRow`:
- Decimal pass-through for OHLCV (asyncpg native NUMERIC).
- `bucket_start` as-is (TIMESTAMPTZ).
- `symbol` + `source` via `str(...)`.

## Hand verification

### §A — Pace algorithm at `pace="max"`

Mock cursor yields 3 rows with bucket_start at 0s, 60s, 120s offsets. `pace="max"` → `pace_factor = inf` → `target_real_dt = sim_dt / inf = 0` → no sleep. All 3 yields complete near-instantly. Test asserts `now_fn` called 1x for replay_start_real on first yield + skipped on subsequent.

### §B — Pace algorithm at `pace="10x"`

Mock cursor yields 2 rows with bucket_start at 0s, 60s offsets. `pace="10x"` → `pace_factor = 10` → for second row: `sim_dt = 60`, `target_real_dt = 6.0s`. Mock `now_fn` returns t=0 then t=2s on second yield; `elapsed = 2`; `sleep_for = max(0, 6 - 2) = 4s`. Test asserts `asyncio.sleep` called once with `~4.0`.

### §C — Pace algorithm at `pace="1x"`

`pace_factor = 1.0`, target_real_dt = sim_dt verbatim. 60s gap = 60s sleep. Test patches asyncio.sleep + verifies 60s argument.

### §D — Decimal preservation

Mock row with `open=Decimal("65000.123456789012")` etc. Test asserts `OHLCRow.open == Decimal("65000.123456789012")` + `isinstance(open, Decimal)`.

### §E — Empty symbols → ValueError

§F — `to_at <= from_at` → ValueError (both equal + reverse).

§G — Invalid source → ValueError (e.g., `source="kraken"`).

§H — Source filter binds to `$4` (verify captured cursor args).
