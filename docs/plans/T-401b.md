## Purpose

`/api/symbol-map/*` admin write endpoints with audit row writes per BRIEF §16.8:2261 ("Write endpoints log every action to `audit_events`"). 5 endpoints — list, get, create, update, delete — with each mutation atomic-coupled to an `audit_events` row inside the same `conn.transaction()`.

T-401b is the **first FastAPI router in the repo to use `pool.acquire() + conn.transaction()` for multi-step writes** (existing transactions live in non-router code: `execution/app/placement.py:304`, `dispatcher.py:129`, `restart.py:148`). Mirror pattern verbatim for atomicity.

T-401b also lands the analytics-api singleton's `now_fn` provider — first time analytics-api needs `datetime.now(UTC)` injection for testability.

```python
@dataclass(frozen=True, slots=True)
class SymbolMapRow:
    """Read projection of symbol_map row (6 columns per §7.2:1131-1138)."""
    input_symbol: str
    canonical_symbol: str
    exchange_source: ExchangeSource  # ExchangeSource (StrEnum)
    notes: str | None
    created_at: datetime
    updated_at: datetime


# Reads (no decorator per existing query module convention)
async def select_all_symbol_map_entries(conn: _DbExecutor) -> list[SymbolMapRow]:
    """Return all symbol_map rows ordered by input_symbol ASC."""

async def select_symbol_map_entry(conn: _DbExecutor, input_symbol: str) -> SymbolMapRow | None:
    """Return one row by input_symbol PK; None if missing."""


# Writes (@non_idempotent per §N3)
@non_idempotent
async def insert_symbol_map_entry(
    conn: _DbExecutor,
    *,
    input_symbol: str,
    canonical_symbol: str,
    exchange_source: str,
    notes: str | None,
    created_at: datetime,
    updated_at: datetime,
) -> SymbolMapRow:
    """INSERT row + RETURN it. Raises asyncpg.UniqueViolationError on duplicate input_symbol PK."""

@non_idempotent
async def update_symbol_map_entry(
    conn: _DbExecutor,
    *,
    input_symbol: str,
    canonical_symbol: str,
    exchange_source: str,
    notes: str | None,
    updated_at: datetime,
) -> SymbolMapRow | None:
    """Full PUT semantics (overwrites canonical_symbol + exchange_source + notes + updated_at).

    `created_at` and PK `input_symbol` preserved. Returns updated row or None if not found.
    """

@non_idempotent
async def delete_symbol_map_entry(conn: _DbExecutor, input_symbol: str) -> bool:
    """DELETE one row; True if deleted, False if not found."""
```

`exchange_source` uses canonical `ExchangeSource(StrEnum)` from `packages.core.types` per WG#1 (consistency with T-401a's `BotStatus`/`ExchangeMode` StrEnum precedent). T-401b adds `ExchangeSource(StrEnum)` to `packages/core/types.py` with 3 members: BINANCE='binance', BYBIT='bybit', CUSTOM='custom'. StrEnum constructor raises ValueError on unknown value — no hand-rolled validator needed.

### `services/analytics_api/app/models/symbol_map.py` (NEW)

```python
class SymbolMapEntryResponse(BaseModel):
    input_symbol: str
    canonical_symbol: str
    exchange_source: ExchangeSource (StrEnum)
    notes: str | None
    created_at: datetime
    updated_at: datetime


class SymbolMapEntryCreate(BaseModel):
    input_symbol: str = Field(..., min_length=1, max_length=64)
    canonical_symbol: str = Field(..., min_length=1, max_length=64)
    exchange_source: ExchangeSource (StrEnum)
    notes: str | None = Field(None, max_length=500)


class SymbolMapEntryUpdate(BaseModel):
    canonical_symbol: str = Field(..., min_length=1, max_length=64)
    exchange_source: ExchangeSource (StrEnum)
    notes: str | None = Field(None, max_length=500)


class SymbolMapListResponse(BaseModel):
    entries: list[SymbolMapEntryResponse]
```

`min_length=1` rejects empty-string spoofing; `max_length=64` matches operational sanity limit (Bybit symbol names are <20 chars in practice; 64 gives safety margin). `notes` `max_length=500` prevents unbounded strings.

### `services/analytics_api/app/routers/symbol_map.py` (NEW)

```python
router = APIRouter(prefix="/api/symbol-map", tags=["symbol-map"])

@router.get("/", response_model=SymbolMapListResponse)
async def list_symbol_map(
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
) -> SymbolMapListResponse:
    """List all symbol_map rows ordered by input_symbol ASC."""

@router.get("/{input_symbol}", response_model=SymbolMapEntryResponse)
async def get_symbol_map_entry(
    input_symbol: str,
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
) -> SymbolMapEntryResponse:
    """Return one entry; 404 if missing."""

@router.post("/", response_model=SymbolMapEntryResponse, status_code=201)
async def create_symbol_map_entry(
    body: SymbolMapEntryCreate,
    request: Request,
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
    now_fn: Annotated[Callable[[], datetime], Depends(get_now_fn)],
) -> SymbolMapEntryResponse:
    """Create one entry. 409 on duplicate PK. Atomic audit row write inside tx."""

@router.put("/{input_symbol}", response_model=SymbolMapEntryResponse)
async def update_symbol_map_entry_endpoint(
    input_symbol: str,
    body: SymbolMapEntryUpdate,
    request: Request,
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
    now_fn: Annotated[Callable[[], datetime], Depends(get_now_fn)],
) -> SymbolMapEntryResponse:
    """Full PUT — overwrites all mutable fields. 404 if missing. Atomic audit."""

@router.delete("/{input_symbol}", status_code=204)
async def delete_symbol_map_entry_endpoint(
    input_symbol: str,
    request: Request,
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
    now_fn: Annotated[Callable[[], datetime], Depends(get_now_fn)],
) -> None:
    """Delete one entry. 404 if missing. Atomic audit."""
```

**Write endpoint contract (load-bearing per §16.8)**: each write opens `async with pool.acquire() as conn, conn.transaction():` (mirror `execution/app/placement.py:304-317` verbatim shape); inside the block:

POST (`create_symbol_map_entry`):
1. `inserted_row = await insert_symbol_map_entry(conn, ...)` (raises UniqueViolationError on duplicate → caught outside tx → 409 Conflict)
2. `await insert_audit_event(conn, action='symbol_map.create', before_state=None, after_state=<asdict(inserted_row)>, ...)`
3. Return `SymbolMapEntryResponse(**asdict(inserted_row))` — outside tx (read-only)

PUT (`update_symbol_map_entry_endpoint`):
1. `before_row = await select_symbol_map_entry(conn, input_symbol)` — pre-read for `before_state`; if None → 404 outside tx
2. `updated_row = await update_symbol_map_entry(conn, ...)` (returns row or None; None means race condition between pre-read and update — extremely unlikely, treated as 404)
3. `await insert_audit_event(conn, action='symbol_map.update', before_state=<asdict(before_row)>, after_state=<asdict(updated_row)>, ...)`
4. Return updated row

DELETE (`delete_symbol_map_entry_endpoint`):
1. `before_row = await select_symbol_map_entry(conn, input_symbol)` — pre-read for `before_state`; if None → 404 outside tx
2. `deleted = await delete_symbol_map_entry(conn, input_symbol)` (True if deleted; False = race → 404)
3. `await insert_audit_event(conn, action='symbol_map.delete', before_state=<asdict(before_row)>, after_state=None, ...)`
4. Return 204 No Content

### Audit row mapping

For each write, audit row populated as:
- `occurred_at`: `now_fn()` — injected callable; tests override
- `actor`: `f"lan:{request.client.host}"` per §16.8:2227 — analytics-api is internal-only (§16.6); fallback `'lan:unknown'` when `request.client is None` (TestClient edge case)
- `action`: `'symbol_map.create'` / `'symbol_map.update'` / `'symbol_map.delete'`
- `entity_type`: `'symbol_map'`
- `entity_id`: `input_symbol` (the PK string)
- `before_state` / `after_state`: per CRUD verb mapping above
- `correlation_id`: from `X-Correlation-ID` request header if present + non-empty, else `None` (T-401b does NOT generate fresh UUIDv4 — keeps T-401b mechanically simple; correlation_id is optional in audit_events per migration 0011 schema; future request-id middleware can fill it in F5+ per §15.2)

## Edge cases

- **POST with input_symbol already present** → caught `asyncpg.UniqueViolationError` outside tx (or inside — UniqueViolation surfaces at INSERT, before audit call) → router returns 409 Conflict; audit row NOT written (transaction rolled back via Python exception propagation)
- **PUT with non-existent input_symbol** → pre-read returns None → router returns 404 BEFORE entering write tx; audit row NOT written
- **DELETE non-existent** → same: pre-read returns None → 404 before tx
- **Pydantic validation failure on POST/PUT body** → FastAPI returns 422 (default); request never reaches business logic; audit row NOT written
- **`X-Correlation-ID` header empty string** → treated as missing (per `header.strip()` check); `correlation_id=None` in audit
- **`request.client is None`** (unusual TestClient state) → `actor='lan:unknown'` rather than raise
- **`notes` field very long** → Pydantic max_length=500 rejects with 422 before SQL
- **Race condition: pre-read row, then external delete, then update** → `update_symbol_map_entry` returns None → router returns 404 (degenerate but correct)
- **Race condition POST: insert raises UniqueViolation but row was inserted by parallel POST** → 409 Conflict; existing row preserved (correct behavior)
- **`get_now_fn` returns naive datetime** (test bug) → asyncpg accepts; audit row `occurred_at` will be UTC-implicit per PG `TIMESTAMPTZ` semantic (assumes session UTC). Test fixture should use `datetime(..., tzinfo=UTC)` per §N1.

## Hand verification

T-401b has no financial math. **math-validator OUT OF SCOPE** per CLAUDE.md Gate 4 conditional list.

The transactional contract has one verifiable property:

**Claim:** Audit row MUST commit atomically with business mutation; failure of either rolls back both.

**Verification:** Each write endpoint opens `async with pool.acquire() as conn, conn.transaction():`; calls business helper (insert/update/delete); calls `insert_audit_event` INSIDE same `with` block; returns response AFTER block exits (response builds from in-tx-captured row but no DB side-effect post-tx). If audit insert raises, asyncpg auto-rolls-back business mutation → both rows absent; router does NOT catch (5xx propagates). Test pins:
- `test_audit_failure_propagates_5xx_when_audit_insert_raises` — monkey-patch `insert_audit_event` to raise; assert response.status_code == 500
- `test_post_symbol_map_returns_409_on_unique_violation` — UniqueViolation at insert before audit call; assert audit NOT called (call_count == 0)
