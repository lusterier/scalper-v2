### POST flow (admin write — atomic audit-tx per §16.8 + T-405 precedent)

Per Gate 1 BLOCKER #1: admin writes MUST emit `audit_events` in the same `conn.transaction()` per §16.8:2261-2263. T-407 mirrors T-405 apply-path's validate-before-tx pattern:

```
1. Pre-validate request body (Pydantic): max_lengths + date range order (mode='after').
2. Parse YAML via load_bot_config_from_string MIMO tx — ValueError → 422 (per OQ-7=A).
3. Check parsed.bot_id == body.bot_id — mismatch → 422 BEFORE tx (mirror T-405 WG#11 spirit).
4. Compute config_hash = sha256(yaml_text.encode("utf-8")).hexdigest()  (raw bytes, no .strip(), per T-405 WG#8).
5. Resolve actor + correlation_id (mirror T-405 _resolve_actor / _resolve_correlation_id).
6. Pre-check bot existence: select_bot_by_id(conn, body.bot_id) MIMO tx — None → 404 "bot {bot_id} not found" (per CONCERN #4 default).
7. Resolve started_at = now_fn() (= queue time per OQ-3=A).
8. Open `async with pool.acquire() as conn, conn.transaction():` — 2-helper same-conn:
     a. inserted_row = await insert_backtest_run(conn, ...)
     b. await insert_audit_event(conn, occurred_at=started_at, actor=..., action='backtest_run.queued',
                                 entity_type='backtest_run', entity_id=str(inserted_row.id),
                                 before_state=None, after_state=_audit_state_dict(inserted_row),
                                 correlation_id=correlation_id, meta={})
9. AFTER tx commits: logger.info("backtest_run.queued", run_id=str(inserted_row.id), ...)
   (per T-405 WG#9 — log post-commit only; on tx rollback, no log).
10. Return 202 + BacktestRunResponse.
```

**Audit `after_state` shape**: per T-405 WG#10 — exclude `config_yaml` field (size discipline). Surviving keys: `id` (UUID str) + `name` + `bot_id` + `config_hash` + `date_range_start` (ISO) + `date_range_end` (ISO) + `status` + `started_at` (ISO) + `finished_at` (None) + `summary` (None) + `notes`. = 11-key shape.

**Audit failure → 5xx**: `insert_audit_event` raise → tx rollback → asyncpg error propagates → FastAPI 500 (mirror T-401b WG#9). NO log emit.

### `BacktestStatus(StrEnum)` (added to `packages/core/types.py`)

```python
class BacktestStatus(StrEnum):
    """backtest_runs.status enum (BRIEF §7.2:1151).

    F4 (T-407): only ``QUEUED`` is ever written by the trigger endpoint.
    F5+ worker transitions ``QUEUED → RUNNING → (COMPLETED|FAILED)``.
    All four values are defined here so the row-narrowing path
    (``BacktestStatus(value)`` in :func:`_row_to_backtest_run`) does NOT
    raise on rows written by a future F5+ worker.
    """

    QUEUED = "queued"
    RUNNING = "running"
    COMPLETED = "completed"
    FAILED = "failed"
```

## Edge cases

1. **Empty list result**: 200 + `{runs: [], total: 0, limit, offset}` (NOT 404 per collection-shape semantics; T-404 WG#8 precedent).
2. **GET unknown UUID**: 404 + `f"backtest run {run_id} not found"` (UUID printed via `str(run_id)`).
3. **GET garbage UUID** (`/api/backtests/not-a-uuid`): FastAPI Path UUID coercion → 422 auto.
4. **POST malformed YAML**: 422 from `load_bot_config_from_string` ValueError catch BEFORE insert (no DB write per T-405 WG#1 validate-before-tx).
5. **POST `bot_id` mismatch (body vs `parsed.bot_id`)**: 422 with detail `f"config_yaml bot_id={parsed.bot_id!r} != request bot_id={body.bot_id!r}"` BEFORE insert.
6. **POST `bot_id` not in `bots` registry** (CONCERN #4 default): 404 with detail `f"bot {body.bot_id!r} not found"` BEFORE tx open (mirror T-405 fail-fast pattern; uses T-401a `select_bot_by_id` helper).
7. **POST `date_range_start >= date_range_end`**: 422 via Pydantic `model_validator(mode='after')`.
8. **POST `date_range_end > now()`**: not validated in F4 (operator may schedule forward-looking placeholder runs; F5+ worker rejects on compute).
9. **`config_yaml` size > 200 KB**: 422 via Pydantic Field max_length.
10. **Audit insert failure inside tx**: tx rollback → both inserts rolled back → asyncpg error → FastAPI 500. `logger.info` NOT emitted (per T-401b WG#9 mirror).
11. **Concurrent UUID collision** (`gen_random_uuid()` 122-bit space): astronomically unlikely; PG raises UniqueViolation if it ever happens, falls through to FastAPI 500.
12. **`?status=garbage` Query param**: FastAPI auto-422 via `BacktestStatus` enum coercion (mirror T-402/T-403/T-404 precedent).
13. **`from_at >= to_at`**: not pre-validated; produces empty list (consistent with T-402 trades range filter precedent).
