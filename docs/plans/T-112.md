### `scripts/backfill_features.py`

```python
"""Backfill feature values from historical OHLC data (§9.3 lines 1515-1518).

Idempotent: re-running with same args yields identical features rows
(T-110b ON CONFLICT DO UPDATE). Property test in
tests/integration/scripts/test_backfill_features_property.py verifies
§9.3 line 1525 determinism.

Usage:
    python scripts/backfill_features.py \
        --feature ind.btcusdt.15m.ema_20 \
        --from 2026-04-01 \
        --to 2026-04-26 \
        [--source binance] \
        [--indicators-yaml configs/features/indicators.yaml] \
        [--database-url postgresql://...]
"""

# imports (no `from services.*` — verified by grep guard at brief-reviewer):
# from packages.features.yaml import INDICATORS_YAML_PATH, load_indicators_yaml
# from packages.features.intervals import INTERVAL_DELTA
# from packages.features.types import OhlcCandle
# from packages.db.queries.feature_engine import fetch_ohlc_range, insert_feature

def _parse_feature_name(feature_name: str) -> tuple[str, str, str]: ...
def _parse_iso_datetime(value: str) -> datetime: ...

async def _backfill_one_feature(
    conn: asyncpg.Connection,
    *,
    feature: Feature,
    feature_name: str,
    symbol: str,
    interval: str,
    source: str,
    from_dt: datetime,
    to_dt: datetime,
    logger: BoundLogger,
) -> int:
    """Iterate OHLC range, compute per candle, upsert. Returns row count.

    Decimal→float seam (verbatim mirror T-110c pipeline._build_update;
    inline because backfill writes via insert_feature kwargs not via
    FeatureUpdate Pydantic):

        value_num_wire = float(fv.value_num) if fv.value_num is not None else None
        value_bool_wire = fv.value_bool
        value_json_wire = (
            {k: float(v) if isinstance(v, Decimal) else v for k, v in fv.value_json.items()}
            if fv.value_json is not None else None
        )
    """


async def _main(args: argparse.Namespace) -> int:
    """Connect, register JSONB codec inline (4 lines), look up feature, backfill."""
    symbol, interval, _rest = _parse_feature_name(args.feature)
    from_dt = _parse_iso_datetime(args.from_)
    to_dt = _parse_iso_datetime(args.to)
    conn = await asyncpg.connect(dsn=args.database_url)
    try:
        # JSONB codec inline (verbatim mirror T-110d _register_jsonb_codec args):
        await conn.set_type_codec(
            "jsonb",
            encoder=json.dumps,
            decoder=json.loads,
            schema="pg_catalog",
        )
        registry = load_indicators_yaml(args.indicators_yaml, [symbol])
        feature: Feature | None = None
        for entries in registry.values():
            for fname, f in entries:
                if fname == args.feature:
                    feature = f
                    break
            if feature is not None:
                break
        if feature is None:
            logger.error("feature_not_in_yaml", feature=args.feature)
            return 1
        await _backfill_one_feature(
            conn, feature=feature, feature_name=args.feature,
            symbol=symbol, interval=interval, source=args.source,
            from_dt=from_dt, to_dt=to_dt, logger=logger,
        )
    finally:
        await conn.close()
    return 0


def main() -> int: ...
```

## Hand verification — backfill compute loop trace

Setup: `--feature ind.btcusdt.15m.ema_20 --from 2026-04-01T00:00:00Z --to 2026-04-26T23:59:59Z`. `ohlc_15m` cagg has 25 days × 96 buckets = 2400 rows.

| step | action                                                                    | observable                                                                |
|------|---------------------------------------------------------------------------|---------------------------------------------------------------------------|
| 1    | argparse → namespace populated                                              | --feature, --from, --to, --source=binance                                 |
| 2    | _parse_feature_name("ind.btcusdt.15m.ema_20") → ("BTCUSDT", "15m", "ema_20") | symbol uppercase canonical                                                 |
| 3    | _parse_iso_datetime("2026-04-01") → datetime(2026, 4, 1, tzinfo=UTC)      | UTC-aware                                                                  |
| 4    | asyncpg.connect(dsn); await conn.set_type_codec("jsonb", ...)              | JSONB codec inline (mirror T-110d args)                                   |
| 5    | load_indicators_yaml(yaml_path, ["BTCUSDT"]) → registry mapping             | T-111 cross-product                                                        |
| 6    | iterate registry; find tuple where feature_name == "ind.btcusdt.15m.ema_20" | match: feature = EmaFeature(20, "15m"), warmup_candles=20                 |
| 7    | await fetch_ohlc_range(conn, symbol="BTCUSDT", interval="15m", source="binance", from_dt=..., to_dt=...) | 2400 rows ASC                                                              |
| 8    | initialise deque(maxlen=20)                                                | empty                                                                      |
| 9    | for each row[i] in 2400:                                                   |                                                                             |
|      | • deque.append(OhlcCandle(symbol=row[0], interval="15m", bucket_start=row[1], open=row[2], …, source=row[7])) | T-110b OhlcRow positional mapping                                          |
|      | • if i >= 19: feature.compute(tuple(deque)) → FeatureValue                  |                                                                             |
|      | • Decimal→float seam: value_num_wire = float(fv.value_num); value_json_wire dict comprehension |                                                                             |
|      | • computed_at = row[1] + INTERVAL_DELTA["15m"]                            | bucket end semantics (T-110c Decision #8)                                  |
|      | • insert_feature(conn, feature_name="ind.btcusdt.15m.ema_20", symbol="BTCUSDT", computed_at=..., value_num=value_num_wire, ...) | row inserted; on re-run, ON CONFLICT DO UPDATE writes through identical values |
| 10   | total rows inserted = 2400 - 19 = 2381                                     | log feature_backfill_complete                                              |
| 11   | await conn.close()                                                         | clean shutdown                                                             |
| 12   | return 0                                                                   | exit code success                                                          |
| 13   | re-run with identical args                                                 | 2381 rows updated in place; PK identical; values identical — §9.3 line 1525 property satisfied |

Critical invariants:
- I1. _parse_feature_name extracts symbol-lowercase from segments[1], canonicalises to uppercase.
- I2. UTC enforcement on --from/--to + DB TIMESTAMPTZ.
- I3. fetch_ohlc_range returns rows ASC by bucket_start.
- I4. Deque maxlen = feature.warmup_candles; under-fill skips compute.
- I5. computed_at = bucket_start + INTERVAL_DELTA[interval] (T-110c Decision #8 inheritance).
- I6. Idempotency: insert_feature ON CONFLICT DO UPDATE on PK — same backfill twice = same DB state.
- I7. Feature.compute deterministic on identical input window (T-107a/b stateless precedent).
- I8. INTERVAL_DELTA reference resolves to `packages.features.intervals.INTERVAL_DELTA` (single source of truth).
- I9. value_json Decimal-aware dict comprehension applied at the wire seam (verbatim from T-110c pipeline._build_update).
