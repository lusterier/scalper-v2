## Purpose

Add `sl_price: Decimal | None` to the `Position` value type so the T-534b SL-watchdog (next leaf, blocked-by this) can ask Bybit "does an SL still exist for this open position" via the existing `get_positions()` read. Structure deliberately mirrors the just-shipped T-530→T-531 (protocol/type-extension leaf → consumer leaf). This is the T-530-shaped leaf — NO caller in T-534a (T-534b is first consumer).

## Type-seam decouple (OQ-5=b)

`Position` (REST snapshot, `get_positions()`) gains `sl_price`; `PositionEvent` (WS stream, `stream_positions()`) does NOT. Rationale (baked here + in the PositionEvent docstring as the durable anchor): T-534b SL-watchdog is a periodic `get_positions()` poll (ADR-0011 §67 H-028) — there is no WS-stream consumer of SL-existence, so per §0.8 ("field lands only when a concrete consumer surfaces") `sl_price` on `PositionEvent` would be a consumer-less field. This is a deliberate first divergence of two previously field-mirrored types; recorded as a plan decision + docstring rationale (NOT a full ADR — single field, one type-seam, T-cluster-scoped; mirror T-530 AccountBalance-without-ADR; §6.7 ADR reserved for cross-cutting / brief-deviation, this is §0.8 compliance, neither).

## Gate-4 — Decimal price decode (no float)

Only numeric concern: Bybit `stopLoss` string → `Decimal(str(sl_raw))`, never `float`. Non-positive/blank (`""`/`"0"`/`"0.00"`/negative) → `None` (Bybit's no-SL sentinel). Paper → `None`. T-534a does NO arithmetic (decode + the `_sl > 0` sign guard only). Mirrors T-530 `Decimal(str(...))` money-decode that math-validator VERIFIED.

## Hand verification

- Bybit item `{"stopLoss": "27000.5", …}` → `_sl = Decimal("27000.5")`, `> 0` → `sl_price = Decimal("27000.5")` (exact, no float).
- `{"stopLoss": ""}` → `_sl = None` → `sl_price = None`. `{"stopLoss": "0"}` → `_sl = Decimal("0")`, not `> 0` → `None`. `{"stopLoss": "0.00"}` → `Decimal("0.00")` not `>0` → `None`.
- Defensive: `{"stopLoss": "-1"}` (never expected) → `Decimal("-1")` not `>0` → `None` (fail-safe; T-534b's consecutive-threshold + transient-skip prevents false emergency_close).
- Paper `get_positions` → every Position `sl_price=None` (watchdog skips paper anyway).
- Divergence note: sibling `avgPrice "0"` is PRESERVED as `Decimal("0")` (recently-closed-flat snapshot, adapter.py:88-90 W#3); `sl_price "0"`→`None` is deliberate (SL@price-0 is semantically impossible = "no SL set"). Documented inline.
- No EMA/seed/division/closed-form — pure decode + sign guard.
