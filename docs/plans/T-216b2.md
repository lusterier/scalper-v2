## Edge cases

- **SL set 3├Ś retry exhaustion** ŌåÆ `emergency_close` (H-004; T-216b1 verbatim).
- **TP set 3├Ś retry exhaustion** (after SL OK) ŌåÆ log ERROR + continue (OQ-2 default A); position open with SL only; T-217 monitor.
- **persist_placement_tx fails** post set_trading_stop calls: tx rollback via `async with`; log ERROR `execution.placement_persist_tx_failed` + return. SL+TP set on Bybit but no DB row ŌåÆ orphan; T-221 reconciles. NO emit (publish-after-persist contract preserved).
- **Paper-mode**: PaperExchange persists paper_* internally; T-218 emits OrderFilled from stream_executions. T-216b2 placement.py path returns after 2 set_trading_stop calls.
- **Dedup ring**: duplicate `(bot_id, signal_id)` ŌåÆ handler not invoked twice (T-216b1 verified). WG#5 malformed-payload WARN (T-216b1 verified).
- **First publish failure in emit_post_commit_events**: WG#6 per-publish try/except ŌåÆ log + continue ŌåÆ second publish still attempted (mirror emergency_close pattern).
