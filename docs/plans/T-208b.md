## OQ resolutions (operator-approved 2026-04-29 — "use defaults")

1. **OQ-1 — Bybit V5 endpoint paths verbatim per docs**: **A**
   - `get_positions(symbol=None)` GET `/v5/position/list` query `{category:"linear", symbol?:str}` retries=3.
   - `get_fill_price(symbol, order_id)` GET `/v5/execution/list` query `{category:"linear", symbol, orderId}` retries=3.
   - `get_closed_pnl_cumulative(sub_account)` GET `/v5/position/closed-pnl` query `{category:"linear", limit:200}` (paginate via `cursor` if response `nextPageCursor` present) retries=3.

2. **OQ-2 — Endpoint-group routing**: **A**
   - `get_positions` → `"positions"`
   - `get_fill_price` → `"orders"` (execution list = order-fill detail)
   - `get_closed_pnl_cumulative` → `"positions"`

3. **OQ-3 — `Position` dataclass mapping**: **A** — per `result.list[]` row map:
   - `symbol = item["symbol"]`
   - `side = "buy"` if `item["side"]=="Buy"`, `"sell"` if `"Sell"`, else `None` (Bybit uses `""` for flat positions per docs).
   - `size = Decimal(item["size"])` (H-015 round-trip — preserve string).
   - `entry_price = Decimal(item["avgPrice"])` if non-empty else `None`.
   - `leverage = int(item["leverage"])` if non-empty else `None`.
   - `unrealized_pnl = Decimal(item["unrealisedPnl"])` if non-empty else `None`.
   - Flat position rows (`size=="0"`) included in return list with `side=None, entry_price=None, leverage=None, unrealized_pnl=None` per T-201 `Position` flat-state semantic (`size==0` + `side=None` = flat).

4. **OQ-4 — `get_fill_price` semantic**: **A** — single GET; if `result.list` empty → return `None`; else `Decimal(result.list[0]["execPrice"])` (LIMIT 1 mirrors T-213c paper-side; first fill chronologically — Bybit V5 docs say result is sorted by execTime DESC by default, but our caller uses this for execution dispatcher cross-check, not chronological audit). H-015 string round-trip preserved.

5. **OQ-5 — `get_closed_pnl_cumulative` semantic**: **A** —
   - Validate `sub_account == self._sub_account`; mismatch raises `ValueError(f"sub_account mismatch: got {sub_account!r}, expected {self._sub_account!r}")` (T-213c precedent).
   - Sum `Decimal(item["closedPnl"])` over all pages of `result.list[]`.
   - Pagination via `cursor` query param if `result.nextPageCursor` non-empty; max `_MAX_CLOSED_PNL_PAGES=10` pages (200 × 10 = 2000 trades cumulative; F2 single-bot scale fits).
   - After 10 pages, log WARN + return partial sum.
   - Returns `Decimal("0")` if no closed trades.

6. **OQ-6 — Pagination inline vs helper**: **A** — inline pagination loop in `get_closed_pnl_cumulative` body; no separate `_paginate` helper. Premature abstraction (single consumer).

7. **OQ-7 — RateLimitError handler**: **A** — every read method `try: client.request → except RateLimitError: await self._on_rate_limit_hit("<group>"); raise`. Existing `_on_rate_limit_hit` from T-208a reused.

8. **OQ-8 — Test strategy**: **A** — unit tests with mock `BybitV5Client.request` returning Bybit V5 envelope shape (`{retCode:0, result:{list:[...], nextPageCursor:"..."}}`); ~15 tests. NO integration tests (deferred to T-222 testnet smoke).

9. **OQ-9 — H-015 round-trip test pin**: **A** — `test_get_positions_preserves_qty_string_through_decimal_round_trip` — input wire `{"size": "0.500000000001"}` → output `Position.size == Decimal("0.500000000001")` (string-exact). Same pattern for `get_fill_price` price.

10. **OQ-10 — Sub_account validation timing**: **A** — validate AT method entry (before `limiter.acquire`); ValueError early-out costs no rate-limit token.

11. **OQ-11 — LOC scope**: **A** — ~150-180 LOC src (3 method bodies + pagination loop ~30 + Position mapper ~20 + sub_account validation ~5 — net of removed 3 stub bodies ~10 LOC). Tests ~250-300 LOC.

## Decisions committed

1. **OQ-defaults applied throughout**; no unresolved decisions remain.

2. **Method-level rate-limit envelope** mirror T-208a:
   ```python
   await self._limiter.acquire(self._sub_account, "<group>")
   try:
       result = await self._client.request("GET", "<path>", params={...}, retries=3)
   except RateLimitError:
       await self._on_rate_limit_hit("<group>")
       raise
   ```
   Reuses existing `_on_rate_limit_hit(group)` from T-208a.

3. **`Position` mapper as private helper** `_map_position_row(item: dict) -> Position`:
   - Reads from Bybit response `result.list[item]` keyed schema per Bybit V5 docs.
   - Returns `Position` per T-201 dataclass shape.
   - Empty-string fields (`""`) handled per Bybit convention: `side==""` → `side=None`; numeric fields `""` → `None`.

4. **Pagination loop in `get_closed_pnl_cumulative`**:
   ```python
   total = Decimal("0")
   cursor: str | None = None
   for page in range(_MAX_CLOSED_PNL_PAGES):
       params = {"category": "linear", "limit": 200}
       if cursor is not None:
           params["cursor"] = cursor
       result = await self._client.request("GET", "/v5/position/closed-pnl", params=params, retries=3)
       for item in result.get("list", []):
           total += Decimal(item["closedPnl"])
       cursor = result.get("nextPageCursor") or None
       if not cursor:
           return total
   logger.warning("bybit_v5.closed_pnl_pagination_capped_at_max_pages", extra={"max_pages": _MAX_CLOSED_PNL_PAGES, "sub_account": sub_account})
   return total
   ```

5. **Module constant**: `_MAX_CLOSED_PNL_PAGES = 10` (F2 single-bot ≤2000 closed trades fits; per §0.8 anti-hypothetical default for F2 scope; bumpable in future via Settings if observability shows truncation).

6. **§N9 evaluation for `_MAX_CLOSED_PNL_PAGES`**: protocol-binding-ish (Bybit limit=200 hardcoded per Bybit V5 docs; max-pages is the safety ceiling tied to that limit). Module constant acceptable per L-001 active control — not operationally tunable per usage pattern (F2 scale never approaches 2000 closed trades; F5+ would refactor to streaming aggregator). Mirror T-205 `_MAX_CAS_RETRIES=3` precedent.

7. **No `metrics_counter` increment for read RateLimitError? — yes, increment**: same handler shape as T-208a writes per Decision #2; T-205 §15.3 deferred counter increments at every adapter rate-limit hit. Reuses T-208a `_on_rate_limit_hit` unchanged.

8. **`get_fill_price` returns first item, not chronologically-first**: Bybit V5 docs say `execution/list` returns sorted by execTime DESC by default; result.list[0] = most recent fill. T-218 dispatcher uses this for execId→order_id cross-check, not for chronological audit. T-213c paper-side `LIMIT 1 ORDER BY executed_at ASC` differs in sort direction but the use case (single-fill order) yields same result regardless. **Note**: if future caller needs first-fill chronologically, add `order` query param. Current usage F2 is single-fill-per-market-order so direction doesn't matter.

9. **`OrderRejected`, `AuthError`, `NetworkTimeout` propagate unhandled**: T-207 client raises these; T-208b read methods don't wrap (mirror T-208a writes — no UnknownState analogue for reads since they're idempotent).

### `packages/exchange/bybit_v5/adapter.py` (extension)

3 method bodies replace `NotImplementedError` stubs from T-208a; 1 new private helper + 1 module constant.

```python
# T-208b additions (replaces T-208a stubs):

_MAX_CLOSED_PNL_PAGES = 10  # F2 scale ceiling (Bybit limit=200 × 10 = 2000 trades).


def _map_position_row(item: dict[str, Any]) -> Position:
    """Map Bybit V5 /v5/position/list result.list[] item → Position dataclass.

    Per OQ-3 default A: Bybit empty-string fields ("") map to None for side
    + numeric fields; non-empty wire strings preserved through Decimal/int
    coercion for H-015 round-trip exactness.
    """
    side_raw = item["side"]
    if side_raw == "Buy":
        side: Literal["buy", "sell"] | None = "buy"
    elif side_raw == "Sell":
        side = "sell"
    else:  # "" (flat) per Bybit V5 convention
        side = None
    return Position(
        symbol=item["symbol"],
        side=side,
        size=Decimal(item["size"]),
        entry_price=Decimal(item["avgPrice"]) if item.get("avgPrice") else None,
        leverage=int(item["leverage"]) if item.get("leverage") else None,
        unrealized_pnl=Decimal(item["unrealisedPnl"]) if item.get("unrealisedPnl") else None,
    )


@idempotent
async def get_positions(self, symbol: str | None = None) -> list[Position]:
    params: dict[str, Any] = {"category": _CATEGORY}
    if symbol is not None:
        params["symbol"] = symbol
    await self._limiter.acquire(self._sub_account, "positions")
    try:
        result = await self._client.request(
            "GET", "/v5/position/list", params=params, retries=3,
        )
    except RateLimitError:
        await self._on_rate_limit_hit("positions")
        raise
    return [_map_position_row(item) for item in result.get("list", [])]


@idempotent
async def get_fill_price(self, symbol: str, order_id: str) -> Decimal | None:
    await self._limiter.acquire(self._sub_account, "orders")
    try:
        result = await self._client.request(
            "GET", "/v5/execution/list",
            params={"category": _CATEGORY, "symbol": symbol, "orderId": order_id},
            retries=3,
        )
    except RateLimitError:
        await self._on_rate_limit_hit("orders")
        raise
    items = result.get("list", [])
    if not items:
        return None
    return Decimal(items[0]["execPrice"])


@idempotent
async def get_closed_pnl_cumulative(self, sub_account: str) -> Decimal:
    if sub_account != self._sub_account:
        raise ValueError(
            f"sub_account mismatch: got {sub_account!r}, expected {self._sub_account!r}"
        )
    total = Decimal("0")
    cursor: str | None = None
    for _page in range(_MAX_CLOSED_PNL_PAGES):
        params: dict[str, Any] = {"category": _CATEGORY, "limit": 200}
        if cursor is not None:
            params["cursor"] = cursor
        await self._limiter.acquire(self._sub_account, "positions")
        try:
            result = await self._client.request(
                "GET", "/v5/position/closed-pnl", params=params, retries=3,
            )
        except RateLimitError:
            await self._on_rate_limit_hit("positions")
            raise
        for item in result.get("list", []):
            total += Decimal(item["closedPnl"])
        cursor = result.get("nextPageCursor") or None
        if not cursor:
            return total
    logger.warning(
        "bybit_v5.closed_pnl_pagination_capped_at_max_pages",
        extra={"max_pages": _MAX_CLOSED_PNL_PAGES, "sub_account": sub_account},
    )
    return total
```

### §F.1 — Bybit V5 wire query snapshots

`get_positions(symbol=None)` → GET `/v5/position/list?category=linear`
`get_positions(symbol="BTCUSDT")` → GET `/v5/position/list?category=linear&symbol=BTCUSDT`
`get_fill_price(symbol="BTCUSDT", order_id="ord-abc")` → GET `/v5/execution/list?category=linear&symbol=BTCUSDT&orderId=ord-abc`
`get_closed_pnl_cumulative(sub_account="sub-a")` page 1 → GET `/v5/position/closed-pnl?category=linear&limit=200`
`get_closed_pnl_cumulative(...)` page 2 → GET `/v5/position/closed-pnl?category=linear&limit=200&cursor=<prev_nextPageCursor>`

### §F.2 — Position dataclass mapping table

| Bybit field | Position field | Conversion |
|-------------|----------------|------------|
| `symbol` | `symbol` | identity |
| `side` | `side` | "Buy"→"buy"; "Sell"→"sell"; ""→None |
| `size` | `size` | `Decimal(str)` (H-015) |
| `avgPrice` | `entry_price` | `Decimal(str)` if non-empty else None |
| `leverage` | `leverage` | `int(str)` if non-empty else None |
| `unrealisedPnl` | `unrealized_pnl` | `Decimal(str)` if non-empty else None |

### §F.3 — Closed-pnl pagination decision tree

1. Page 0: GET with no cursor → `result = {list: [...], nextPageCursor: "abc"}`. Sum list. cursor="abc".
2. Page 1: GET with cursor=abc → `result = {list: [...], nextPageCursor: ""}`. Sum list. cursor=None.
3. Loop exit: cursor is None → return total.

OR (cap path):

1-9. Pages 0-9: each returns nextPageCursor non-empty.
10. Page 10 attempted but loop counter exhausted (range(10) = pages 0-9; page 10 not entered). WARN log. Return partial total.

Test `test_get_closed_pnl_cumulative_caps_at_max_pages_with_warn_log`: mock returns 10 consecutive pages each with nextPageCursor="next" → loop runs 10 times → exits range without natural termination → WARN log emitted with extras `{max_pages: 10, sub_account: "sub-a"}`.

### §F.4 — H-015 round-trip preservation

Decimal exactness from wire string to Position dataclass:
- Wire: `{"size": "0.500000000001"}` (12 fractional digits per §5.13 NUMERIC(30,12)).
- Python: `Decimal("0.500000000001")` (exact; 12 fractional digits preserved).
- No `float()` coercion anywhere in the read path.

Test pin asserts `position.size == Decimal("0.500000000001")` and `isinstance(position.size, Decimal)` and `str(position.size) == "0.500000000001"` (W#5 type pin from T-208a precedent extended to read side).

## Lifecycle

`BybitV5Adapter` lifecycle (T-208b scope) — read-only, stateless across calls:

1. `get_positions(symbol)`: validate (none) → limiter.acquire("positions") → GET → map response.list[] → return.
2. `get_fill_price(symbol, order_id)`: limiter.acquire("orders") → GET → first result.list[0] price OR None.
3. `get_closed_pnl_cumulative(sub_account)`: validate sub_account match → loop pages → limiter.acquire per page → GET with cursor → sum closedPnl → return on cursor=None or after MAX_PAGES.

## Edge cases

1. **Bybit returns `result.list` missing/null**: `result.get("list", [])` defaults to empty list; method returns empty/None per OQ-1.

2. **`size==0` row WITHOUT explicit symbol filter**: Bybit V5 `/v5/position/list` without symbol filter returns only NON-flat positions per docs. `size==0` row only appears WITH symbol filter (operator queried specific symbol). Test pin `test_get_positions_maps_flat_row_with_empty_side_to_None_side` documents this.

3. **`avgPrice=""` for flat position**: handled by `if item.get("avgPrice")` short-circuit → entry_price=None.

4. **Pagination capped at 10 pages**: WARN log emitted; partial sum returned. Acceptable per OQ-5 default A. If T-220 P&L audit observes truncation, T-F2+ task to refactor to streaming aggregator.

5. **`get_fill_price` for non-existent order_id**: Bybit returns `result.list=[]` → return None per OQ-4.

6. **`get_closed_pnl_cumulative` for never-traded sub_account**: Bybit returns `result.list=[]` page 1, `nextPageCursor=""` → loop returns `Decimal("0")`.

7. **Sub_account validation error path**: ValueError raised BEFORE `limiter.acquire` per OQ-10 default A — no token consumed on caller mistake.
