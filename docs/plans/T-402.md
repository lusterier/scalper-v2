## Purpose

Two read-only endpoint groups for the dashboard's live-view + trade-explorer sections:

1. **`/api/positions/*`** — open positions across all bots from `position_state` table (BRIEF §9.6:1623 + §14.3:2061). Single endpoint with optional `?bot_id=` filter; collection bounded at <50 rows across active bots.
2. **`/api/trades/*`** — trade history paginated/filtered (list) + per-trade detail (BRIEF §9.6:1624 + §14.3:2062). Two endpoints: paginated list with filters + detail by id.

T-402 is **read-only** (no admin writes); audit emission N/A. Pagination + filter introduced for `/api/trades/` per §9.6 ("REST for paginated queries"); positions skipped per OQ-4 default (collection <50 rows).

Per §0.8 anti-hypothetical: drill-down full timeline orchestration (signal + scoring + executions + orders + shadow variants + post-close snapshots per BRIEF §14.3:2062) is **T-414 UI concern**, NOT backend join. T-402 ships granular trade endpoint; UI joins client-side via T-403/T-404/T-407 endpoint groups.

    mfe_pct: float | None
    mae_pct: float | None
    confidence_score: float | None
    meta: dict[str, Any]


async def select_open_positions(
    conn: _DbExecutor,
    *,
    bot_id: str | None = None,
) -> list[OpenPositionRow]:
    """Return all rows from position_state, optionally filtered by bot_id.

    ORDER BY bot_id, symbol for deterministic UI ordering. position_state
    only contains OPEN positions by definition (T-219 deletes on close);
    no status filter needed.
    """


async def select_trades_paginated(
    conn: _DbExecutor,
    *,
    bot_id: str | None,
    symbol: str | None,
    status: str | None,        # 'open' | 'closed' | 'error'
    from_at: datetime | None,  # closed_at >= from_at (or opened_at if status='open')
    to_at: datetime | None,    # closed_at < to_at
    limit: int,
    offset: int,

    """Return one page of trades with optional filters.

    ORDER BY closed_at DESC NULLS FIRST for "most recent first" with open
    trades floating to top. Limit/offset clamped by caller (router enforces
    1 ≤ limit ≤ 200; 0 ≤ offset).

async def count_trades(
    conn: _DbExecutor,
    *,
    bot_id: str | None,
    symbol: str | None,
    status: str | None,
    from_at: datetime | None,
    to_at: datetime | None,
) -> int:
    """Total count of trades matching filters (for pagination envelope)."""


async def select_trade_by_id(
    conn: _DbExecutor,
    trade_id: int,
) -> TradeRow | None:
    """Return one trade row by PK; None if not found."""

### `services/analytics_api/app/routers/positions.py` (NEW)

```python
router = APIRouter(prefix="/api/positions", tags=["positions"])


@router.get("/", response_model=OpenPositionListResponse)
async def list_open_positions(
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
    bot_id: str | None = Query(default=None, description="Filter to one bot's positions"),
) -> OpenPositionListResponse:
    """Return all open positions across bots, or filter to one bot."""
    async with pool.acquire() as conn:
        rows = await select_open_positions(conn, bot_id=bot_id)
    return OpenPositionListResponse(
        positions=[OpenPositionResponse(**asdict(r)) for r in rows],
    )
```

### `services/analytics_api/app/routers/trades.py` (NEW)

```python
router = APIRouter(prefix="/api/trades", tags=["trades"])

_DEFAULT_LIMIT = 50
_MAX_LIMIT = 200


@router.get("/", response_model=TradeListResponse)
async def list_trades(
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
    bot_id: str | None = Query(default=None),
    symbol: str | None = Query(default=None),
    status: str | None = Query(default=None, description="open | closed | error"),
    from_at: datetime | None = Query(default=None, alias="from"),
    to_at: datetime | None = Query(default=None, alias="to"),
    limit: int = Query(default=_DEFAULT_LIMIT, ge=1, le=_MAX_LIMIT),
    offset: int = Query(default=0, ge=0),
) -> TradeListResponse:
    """List trades paginated + filtered. ORDER BY closed_at DESC NULLS FIRST."""
    async with pool.acquire() as conn:
        rows = await select_trades_paginated(
            conn,
            bot_id=bot_id, symbol=symbol, status=status,
            from_at=from_at, to_at=to_at,
            limit=limit, offset=offset,
        )
        total = await count_trades(
            conn,
            bot_id=bot_id, symbol=symbol, status=status,
            from_at=from_at, to_at=to_at,
        )
    return TradeListResponse(
        trades=[TradeResponse(**asdict(r)) for r in rows],
        total=total, limit=limit, offset=offset,
    )


@router.get("/{trade_id}", response_model=TradeResponse)
async def get_trade(
    trade_id: int,
    pool: Annotated[asyncpg.Pool, Depends(get_pool)],
) -> TradeResponse:
    """Return one trade by PK; 404 if missing."""
    async with pool.acquire() as conn:
        row = await select_trade_by_id(conn, trade_id)
    if row is None:
        raise HTTPException(404, detail=f"trade {trade_id} not found")
    return TradeResponse(**asdict(row))
```

## Edge cases

- **`?bot_id=missing`** → empty positions list (200, NOT 404 — no entity-exists semantic on filter param)
- **`?status=garbage`** → SQL filter passes through as-is; 0 rows match; 200 with empty `trades`. **No 422 validation** — would require StrEnum + introduce more places-of-failure for a string column. Per BRIEF §7.2:1003 status is just a TEXT enum at DB layer. Operator-acceptable trade-off; T-414 UI controls choice via dropdown.
- **`?from=<future>&to=<past>`** → 0 rows; 200 with empty trades (degenerate but mathematically correct)
- **`?limit=0`** → 422 from FastAPI Query(ge=1)
- **`?limit=201`** → 422 from FastAPI Query(le=200)
- **Trade with `realized_pnl=null`** (open trade) → JSON `null` in response (Pydantic Optional)
- **Position with `mfe_price=null`** (no fill movement yet) → JSON `null`
- **Decimal `'0E-12'` from PG** (asyncpg returns canonicalised Decimals) → Pydantic str repr `'0E-12'` not `'0.000000000000'`. Test asserts string presence + parseability via `Decimal(value_str)`, not exact string match (defensive against PG decimal canonicalisation).
- **Empty `meta` JSONB** → returns `{}` (asyncpg JSONB codec from T-401a's main.py init)
