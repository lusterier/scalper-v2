## Goal

Refactor `packages/exchange/paper/adapter.py` PaperExchange to support shadow-mode lifecycle (BRIEF §13.3 "lightweight position FSM running against PaperExchange seeded with live entry"). 4 specific extensions, all backwards-compat with live + replay modes:

1. **`seed_open_state: dict | None = None` constructor kwarg** — when provided, populates `_active_positions[symbol]` + `_last_price[symbol]` + `_last_candle[symbol]` from supplied entry context BEFORE first candle arrives. Eliminates race where `place_market_order` raises `ValueError("No last-observed price for ...")` when called immediately after `start_consuming()`.
2. **`bus_unsubscribe_market_ohlc()` method** — track `Subscription` handle returned by `await self._bus.subscribe(...)` in `start_consuming()`; expose async unsubscribe (NATS path) OR sets `subscription.active = False` (ReplayBus path). Required by T-511b shadow-worker for clean per-variant teardown post-terminal (H-016 hazard ergonomic precondition).
3. **`sl_type` field on `_active_positions` dict** — new per-symbol state field tracking SL semantic (`Literal["protective", "be", "trail"]` 3-state vocabulary mirror live execution-service): `'protective'` initial; `'trail'` after partial-TP promotion per **ADR-0005 v2** verbatim mandate (`partial_tp → sl_type='trail'` per ADR-0005 line 28-29 + dispatcher.py:213-215 implementation). T-511a writes `'protective'` (in `set_trading_stop`) + `'trail'` (in `_drain_sl_tp_fill` partial_tp path). The third value `'be'` comes from a separate **lifecycle BE-trigger path** (execution-service `lifecycle.py:166-198` — NOT partial_tp; price crosses `be_trigger` config); PE has no lifecycle BE-trigger today. T-511b shadow worker BE_HIT outcome handling: deferred to T-511b plan stage (PE+lifecycle BE-trigger refactor would be separate concern; for T-511a v1 PE writes only protective/trail). Enables T-511b SL_HIT vs TRAIL_HIT disambiguation; BE_HIT mapping flagged as T-511b open question.
4. **Drop `closed_pnl_lock` ctor kwarg artifact** — T-511 plan §"Out of scope" mistakenly mentioned `closed_pnl_lock=asyncio.Lock()` on PaperExchange; PE constructor never had this param. T-511a explicitly documents that PaperExchange does NOT need `closed_pnl_lock` (single-process per-instance state) — no constructor change, just docstring clarification.

Per OQ-2=A (T-511 plan cycle): `sl_type` field added to PE _active_positions dict (NOT side-state in shadow worker). Matches live execution-service convention; clean BE_HIT vs SL_HIT disambiguation.

### `_apply_seed_open_state()` private helper

```python
def _apply_seed_open_state(self) -> None:
    """T-511a: pre-populate caches from seed_open_state for shadow-mode lifecycle.

    Eliminates the place_market_order race where _last_price[symbol] is None
    until first market.ohlc.1m candle arrives (typical 0..60s window between
    start_consuming() and first candle tick).

    Synthesised _last_candle uses entry_price for all 4 OHLCV fields (degenerate
    candle representing the entry instant); _compute_slippage reads only high/low
    which equal entry_price → zero slippage on synthetic candle (correct for
    shadow seeding — variant inherits parent's already-applied slippage in
    entry_price). Mirrors T-506 BLOCKER #1 fix at adapter.py:418-427 — schema lie
    contained at cache boundary (source='binance' Literal default; _compute_slippage
    never reads candle.source — verified by grep at T-506 plan-time).
    """
    s = self._seed_open_state
    assert s is not None  # caller guard
    symbol = s["symbol"]
    self._active_positions[symbol] = {
        "trade_id": int(s["trade_id"]),
        "side": s["side"],
        "qty": Decimal(s["qty"]),
        "entry_price": Decimal(s["entry_price"]),
        "entry_fee": Decimal(str(s.get("entry_fee", "0"))),
        "fees_paid": Decimal(str(s.get("fees_paid", "0"))),
        "sl_price": Decimal(s["sl_price"]) if s.get("sl_price") is not None else None,
        "tp_price": Decimal(s["tp_price"]) if s.get("tp_price") is not None else None,
        "tp_size": Decimal(s.get("tp_size", s["qty"])),
        "tpsl_mode": s.get("tpsl_mode", "Full"),
        "tp_hit": bool(s.get("tp_hit", False)),
        "sl_type": s.get("sl_type", "protective"),  # T-511a: track SL semantic
    }
    entry = Decimal(s["entry_price"])
    self._last_price[symbol] = entry
    self._last_candle[symbol] = OhlcCandlePayload(
        symbol=symbol,
        bucket_start=s.get("opened_at", self._now_fn()),
        open=entry, high=entry, low=entry, close=entry,
        volume=Decimal("0"),
        is_closed=True,
    )
```

### `bus_unsubscribe_market_ohlc()` method

```python
async def bus_unsubscribe_market_ohlc(self) -> None:
    """T-511a: idempotent unsubscribe from market.ohlc.1m.> subject.

    Required by T-511b shadow worker per-variant teardown post-terminal so
    orphan _on_candle handlers don't keep firing after variant completes
    (H-016 hazard ergonomic precondition).

    Implementation depends on subscription type:
    - NATS Subscription (live mode): call subscription.unsubscribe() (async)
    - ReplaySubscription (T-502): set subscription.active = False

    Idempotent: second call no-op.
    """
    sub = self._market_ohlc_subscription
    if sub is None:
        return
    self._market_ohlc_subscription = None
    if hasattr(sub, "unsubscribe") and callable(sub.unsubscribe):
        await sub.unsubscribe()
    elif hasattr(sub, "active"):
        sub.active = False
```
