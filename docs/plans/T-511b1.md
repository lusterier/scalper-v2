## Goal

Ship the shadow-worker FSM core: per-variant async task body that runs `PaperExchange` (T-511a refactored) seeded with live entry, drives BE-trigger / trail-SL adjustments via PE primitives, observes terminal via `terminal_callback`, classifies outcome per ADR-0005 v2 H-024 derivation chain, and persists open + terminal rows to `shadow_variants` (migration 0014). H-016 finalizer cleanup on per-variant exception escape.

5 ADR-0005 v2 outcomes: `SL_HIT` (sl_type='protective' at close), `BE_HIT` (sl_type='be'), `TP_TRAIL` (sl_type='trail' post-partial_tp), `TP_FULL` (full TP cross), `TIMEOUT` (max_duration_hours elapsed; no SL/TP cross).

3-state PE vocabulary (T-511a-shipped): `protective` initial / `be` (T-511b1 writes via new `set_trading_stop` `sl_type` kwarg) / `trail` (post-`_drain_partial_tp` per ADR-0005 v2; T-511a-shipped). Shadow worker terminal classification reads `sl_type_at_close` snapshot per ADR-0005 v2 H-024 invariant ‚Äî mirror live `dispatcher._derive_exec_type` `dispatcher.py:316-380`.

## Hand verification (BE-trigger + trail SL math)

Long entry at 65000, qty=1.0, fees=0 for arithmetic clarity.

**Variant `baseline` (be_trigger=0.005, be_sl_level=0.001, trail_pct=0.003)**:
- BE trigger threshold: `(current - 65000) / 65000 >= 0.005` ‚Üí trigger price = `65000 * 1.005 = 65325`
- BE SL price (after trigger): `65000 * (1 + 0.001) = 65065`
- Candle high=65325 ‚Üí `_check_be_trigger=True` ‚Üí shadow worker calls `pe.set_trading_stop(symbol="BTCUSDT", tpsl_mode="Full", sl_price=Decimal("65065"), sl_type="be")`
- Reverse candle low=65060 < 65065 ‚Üí SL cross ‚Üí terminal_callback fires with `exec_type='sl'`, `sl_type_at_close='be'` ‚Üí outcome=`BE_HIT`
- Realized PnL: `(65065 - 65000) * 1.0 = +65`

**Variant `no_be` (be_trigger=0; never triggers BE)**:
- Identical price path; BE never fires; sl_type stays `'protective'`
- Reverse candle low=64500 (initial protective SL=64500 from `pe.set_trading_stop(tpsl_mode="Full", sl_price=Decimal("64500"))` at startup) ‚Üí SL cross ‚Üí outcome=`SL_HIT`
- Realized PnL: `(64500 - 65000) * 1.0 = -500`

**Variant `full_tp` (tp_qty_pct=1.0, no trail)**:
- Forward candle high=66000 (initial TP=66000) ‚Üí full TP cross ‚Üí outcome=`TP_FULL`
- Realized PnL: `(66000 - 65000) * 1.0 = +1000`

**Variant after partial_tp (tp_qty_pct=0.2, trail_pct=0.003)**:
- Forward candle high=66000 ‚Üí `_drain_partial_tp` fires (T-511a-shipped) ‚Üí 0.2 qty closed at 66000 + sl_type promoted to `'trail'`
- Best price tracker = 66000; trail SL = `66000 * (1 - 0.003) = 65802`
- Reverse candle low=65800 < 65802 ‚Üí SL cross ‚Üí terminal_callback fires with `exec_type='sl'`, `sl_type_at_close='trail'` ‚Üí outcome=`TP_TRAIL`
- Realized PnL on residual: `(66000 - 65000) * 0.2 + (65802 - 65000) * 0.8 = 200 + 641.6 = +841.6`
