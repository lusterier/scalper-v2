## Goal

Ship the shadow-variant **restart-recovery infrastructure** — on `execution-service` startup, scan `shadow_variants WHERE terminated_at IS NULL`, for each active variant: (a) check parent trade still open (live `trades` for `parent_kind='live'`; paper `paper_trades` for `parent_kind='paper'`); (b) replay OHLC from `created_at` to `now()` using same `_step` body as live (via T-511b1's `_make_candle_handler` + PE `_on_candle` intra-candle path); (c) terminal detection via `terminal_future.done()` post-each-candle (handles partial-TP-then-SL scenario per H-024 v2 — `_active_positions[symbol]` absence is full-close-only signal and would miss the partial-TP path); (d) if terminal during replay → finalize; otherwise → spawn live continuation. Per BRIEF §13.4 + §13.7 invariants.

**Variant overrides persistence**: T-511b1 shipped `_run_shadow_variant` does NOT persist overrides to DB — they live only in the in-memory `ShadowStartPayload.variants[i].overrides` dict. T-512a requires retro-fit: at insert time, write `meta={"overrides": {k: str(v) for k, v in variant.overrides.items()}}` to `shadow_variants.meta` JSONB column (Decimal→str pre-stringification handled by `_serialize_meta` + `_to_jsonable` per L-013 convention). Resume path decodes via `Decimal(row.meta["overrides"][k])`. Required-effect retro-fit (analogous to T-511b2a 1-LOC propagation pattern) — without it, resume path cannot reconstruct PE `seed_open_state` (sl_price/tp_price/tp_size) or `_make_candle_handler` factory args (be_trigger/be_sl_level/trail_pct).

    bus: BusProtocol,
    settings: Settings,

    Enumerates all active variants across bots; for each, dispatches a
    background asyncio.Task running ``replay_shadow_variant_to_now``.

    so the parent-close H-016 hook (T-511b2) can cancel them if the parent

Internals:

- `_check_parent_open(conn, row) -> bool` — SQL inline: `SELECT status FROM trades WHERE id=$1` for live OR `SELECT status FROM paper_trades WHERE id=$1` for paper; closed semantic varies (`status='closed'` for live; `closed_at IS NOT NULL` for paper). Returns True if parent still active.

## Hand verification

### Wall-clock carry-over math (OQ-3=A)

Variant created at `T_0 = 2026-05-08 12:00:00 UTC` with `max_duration_hours = 4`. Restart happens at `T_now = 2026-05-08 13:50:00 UTC` (110 min after creation; assume 50 min server downtime, 60 min uptime before downtime).

- Replay window: `T_0` → `T_now` = 110 minutes (110 candles at 1m cadence).
- Expires at: `T_0 + 4h = 2026-05-08 16:00:00 UTC`.
- Time remaining post-replay: `expires_at - T_now = 16:00 - 13:50 = 2h 10min = 7800 seconds`.
- If replay terminates mid-window (e.g. SL cross at T_0 + 30min = 12:30) → finalize as `sl_hit` (replay-derived terminal). No live continuation.
- If replay does NOT terminate by T_now → spawn live continuation; `asyncio.wait_for(terminal_future, timeout=7800)` for live phase.

### Replay determinism (BRIEF §13.7 unit test)

Given variant `(side='buy', entry_price=65000, sl_pct=0.005, tp_pct=0.01, be_trigger=0.005, be_sl_level=0.001)` + canned 5-candle replay window:

- T+0: open=65000, high=65000, low=65000, close=65000 → no SL/TP cross; sl_type stays 'protective'.
- T+1: open=65100, high=65325, low=65050, close=65300 → BE-trigger fires at high=65325 (favorable_pct=0.005); shadow worker calls `pe.set_trading_stop(sl_price=65065, sl_type='be')`.
- T+2: open=65300, high=65500, low=65200, close=65400 → no SL cross (low > 65065); sl_type stays 'be'.
- T+3: open=65400, high=65450, low=65060, close=65200 → SL cross at low=65060 < 65065; PE `_drain_full_close` fires; terminal_event with sl_type_at_close='be' → outcome=`BE_HIT`, realized_pnl = (65065 - 65000) × qty.
- T+4 (would not be processed): replay loop terminates after T+3.

Same input → same outcome on re-replay. Pure-function path (Decimal arithmetic + state machine).

### Intra-candle path equivalence (BRIEF §13.7 unit test)

Live path: PE._on_candle(candle) → _check_sl_tp_crosses(candle) intra-candle → _drain_sl_tp_fill if cross.
Replay path: same PE._on_candle(candle) called directly with cursor-iterated candle envelope → same intra-candle detection.

Test: construct identical canned candle stream; route once through live PE.start_consuming() (fed by bus mock) and once through replay direct-invoke; assert resulting `_active_positions` state + ExecutionEvent.qty/price match byte-for-byte.
