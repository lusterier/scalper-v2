## Goal

Add `--compare run_A_uuid run_B_uuid` mode to `scripts/backtest.py` per BRIEF §12.2:1969-1970:

> `python scripts/backtest.py --compare run_A_uuid run_B_uuid`: outputs a diff of aggregate metrics and a per-trade diff where the same signal produced different outcomes.

Pure read-only CLI (no DB writes). Stdout text-table output (per OQ-1=A; JSON deferred per §0.8).

End-to-end FSM:
1. argparse — `--compare A B` flag (nargs=2); validate UUIDs + A != B
2. Open asyncpg pool (read-only)
3. Select `backtest_runs.summary` JSONB for both run_ids — fail SystemExit(1) if either missing
4. Compute aggregate diff (5 stats × 2 runs + delta column)
5. SELECT JOIN backtest_trades on signal_id WHERE close_reason or realized_pnl differs (per OQ-2=A) — both runs must have rows for same signal
6. Format text table to stdout
7. Exit 0

### Exit codes (compare mode)

- `0` — comparison output written to stdout
- `1` — runtime failure (e.g., DB connection error, run_id not found)
- `2` — argparse / validation error (e.g., same run_id twice; mode flags conflict)

### Stdout output format (per OQ-1=A text-table)

```
Backtest comparison: A=<run_a_uuid_short> vs B=<run_b_uuid_short>
================================================================

Aggregate metrics:
  Metric          A             B             Δ (B-A)
  --------------  -----------   -----------   -----------
  total_trades    4             4             0
  wr              0.5000        0.7500        +0.2500
  pnl             22.00         45.00         +23.00
  pf              3.7500        9.0000        +5.2500
  mdd             5.00          2.00          -3.00

Per-trade differences (3 of 4 common signals diverged):
  signal_id   A.close_reason   A.realized_pnl   B.close_reason   B.realized_pnl
  ----------  ---------------  ---------------  ---------------  ---------------
  101         sl               -5.00            tp               10.00
  102         tp               10.00            tp               15.00
  104         sl               -3.00            tp               20.00
```

Edge cases:
- PF=None per ADR-0008 → display `—` (em-dash) in table; delta column shows `n/a`
- Empty per-trade diff → "No diverging trades found." line; no table
- Run with `total_trades=0` (no trades) → aggregate table still rendered with `0` / `null` rows

### `cli_main()` dispatch update

```python
def cli_main() -> int:
    parser = _build_parser()
    args = parser.parse_args()
    if args.compare:
        # Compare-mode validation.
        if args.compare[0] == args.compare[1]:
            sys.stderr.write("--compare run_A_uuid must differ from run_B_uuid\n")
            return 2
        return asyncio.run(main_compare(args))
    # Run-mode validation (existing T-507b path).
    if not (args.bot and args.from_at and args.to_at and args.config_path):
        sys.stderr.write(
            "Run mode requires --bot, --from, --to, --config-path "
            "(or use --compare for comparison mode)\n"
        )
        return 2
    args.from_at = _parse_utc_datetime("from", args.from_at)
    args.to_at = _parse_utc_datetime("to", args.to_at)
    if args.to_at <= args.from_at:
        sys.stderr.write("--to must be > --from\n")
        return 2
    return asyncio.run(main(args))
```

### `main_compare(args)` flow

```python
async def main_compare(args: argparse.Namespace) -> int:
    """Composition root for --compare mode (read-only)."""
    configure(level="INFO")
    system_logger = get_logger("backtest", "system")

    db_url = args.db_url or _require_env("DATABASE_URL")
    pool = await create_pool(db_url, application_name="backtest-cli-compare")
    try:
        run_a_id = _parse_uuid("run_A_uuid", args.compare[0])
        run_b_id = _parse_uuid("run_B_uuid", args.compare[1])
        async with pool.acquire() as conn:
            summary_a = await select_backtest_run_summary(conn, run_id=run_a_id)
            summary_b = await select_backtest_run_summary(conn, run_id=run_b_id)
            if summary_a is None:
                sys.stderr.write(f"run_id {run_a_id} not found in backtest_runs\n")
                return 1
            if summary_b is None:
                sys.stderr.write(f"run_id {run_b_id} not found in backtest_runs\n")
                return 1
            diverging = await select_diverging_trades_for_compare(
                conn, run_a=run_a_id, run_b=run_b_id,
            )

        # 4. Format + emit.
        sys.stdout.write(
            f"Backtest comparison: A={str(run_a_id)[:8]} vs B={str(run_b_id)[:8]}\n"
        )
        sys.stdout.write("=" * 64 + "\n\n")
        sys.stdout.write(_format_aggregate_diff(summary_a, summary_b))
        sys.stdout.write("\n")
        sys.stdout.write(_format_per_trade_diff(diverging))
        sys.stdout.write("\n")
        return 0

    except Exception as exc:
        system_logger.error("backtest.compare_failed", error=str(exc), exc_info=True)
        return 1
    finally:
        await pool.close()
```

### `_format_aggregate_diff(summary_a, summary_b)`

Builds 8-line text table. PF=None handled per ADR-0008: display `"—"`; delta column `"n/a"` if either side None.

Delta computation:
- `total_trades`: int delta `b - a`
- `wr`: float delta `b - a` (or `n/a` if either None)
- `pnl`: Decimal delta `Decimal(b) - Decimal(a)` → `str` for table
- `pf`: float delta `b - a` (or `n/a` per ADR-0008)
- `mdd`: Decimal delta `Decimal(b) - Decimal(a)` → `str`

### `_format_per_trade_diff(rows)`

Returns multi-line string. Header line + data rows (1 row per diverging trade). Row count summary line `"Per-trade differences (N of M common signals diverged):"`. M = total common signal_ids (computed in helper).

For empty `rows`: returns `"No diverging trades found.\n"`.

## SQL helpers

### `select_backtest_run_summary` (NEW)

```python
async def select_backtest_run_summary(
    conn: _DbExecutor, *, run_id: UUID,
) -> dict[str, Any] | None:
    """Return parsed `backtest_runs.summary` JSONB for run_id, or None if missing.

    Read-only; no @idempotent decorator needed (read by definition replay-safe;
    CLI helpers without side effects don't carry annotation per existing
    convention at packages/db/queries/analytics.py:1741 select_backtest_run_by_id).

    Returns None on no row (operator passed wrong UUID or run was deleted);
    caller (CLI) emits SystemExit(1) with operator-actionable message.

    L-013 read-side N/A: CLI doesn't register JSONB codec; asyncpg returns
    summary as `str` (text-mode); helper parses via `json.loads` to return
    dict per public API contract.
    """
    row = await conn.fetchrow(
        "SELECT summary FROM backtest_runs WHERE id = $1", run_id,
    )
    if row is None or row["summary"] is None:
        return None
    summary = row["summary"]
    if isinstance(summary, str):
        return cast(dict[str, Any], json.loads(summary))
    # Codec-registered consumer (analytics-api) would return dict directly.
    return cast(dict[str, Any], summary)
```

### `select_diverging_trades_for_compare` (NEW)

```python
@dataclass(frozen=True, slots=True)
class DivergingTradeRow:
    """Per-trade diff row: signal_id + (close_reason, realized_pnl) for both runs."""

    signal_id: int
    a_close_reason: str | None
    a_realized_pnl: Decimal | None
    b_close_reason: str | None
    b_realized_pnl: Decimal | None


_SELECT_DIVERGING_TRADES_SQL = """
    SELECT
        a.signal_id,
        a.close_reason AS a_close_reason,
        a.realized_pnl AS a_realized_pnl,
        b.close_reason AS b_close_reason,
        b.realized_pnl AS b_realized_pnl
    FROM backtest_trades a
    INNER JOIN backtest_trades b ON a.signal_id = b.signal_id
    WHERE a.run_id = $1
      AND b.run_id = $2
      AND a.signal_id IS NOT NULL
      AND (
          COALESCE(a.close_reason, '') != COALESCE(b.close_reason, '')
          OR a.realized_pnl IS DISTINCT FROM b.realized_pnl
      )
    ORDER BY a.signal_id
"""


async def select_diverging_trades_for_compare(
    conn: _DbExecutor, *, run_a: UUID, run_b: UUID,
) -> list[DivergingTradeRow]:
    """SELECT trades JOIN by signal_id where close_reason OR realized_pnl differs.

    Per OQ-2=A: diff scope is close_reason + realized_pnl ONLY (mfe/mae/
    confidence_score noise excluded). Decimal exact equality (no tolerance).
    `IS DISTINCT FROM` handles NULL semantics correctly (NULL != NULL → TRUE
    in normal SQL; IS DISTINCT FROM treats NULL == NULL).

    Read-only; no annotation. Returns list of DivergingTradeRow ordered by
    signal_id ascending.
    """
    rows = await conn.fetch(_SELECT_DIVERGING_TRADES_SQL, run_a, run_b)
    return [
        DivergingTradeRow(
            signal_id=int(r["signal_id"]),
            a_close_reason=r["a_close_reason"],
            a_realized_pnl=(
                Decimal(r["a_realized_pnl"])
                if r["a_realized_pnl"] is not None else None
            ),
            b_close_reason=r["b_close_reason"],
            b_realized_pnl=(
                Decimal(r["b_realized_pnl"])
                if r["b_realized_pnl"] is not None else None
            ),
        )
        for r in rows
    ]
```

## Hand verification

### §A — Aggregate diff (4 trades each side)

Fixture A summary: `{total_trades:4, wr:0.5, pnl:"22.00", pf:3.75, mdd:"5.00"}`
Fixture B summary: `{total_trades:4, wr:0.75, pnl:"45.00", pf:9.0, mdd:"2.00"}`

Hand-computed deltas:
- total_trades: 4 - 4 = **0**
- wr: 0.75 - 0.5 = **+0.25** (rounded to 4 decimals)
- pnl: Decimal("45.00") - Decimal("22.00") = **+23.00**
- pf: 9.0 - 3.75 = **+5.25**
- mdd: Decimal("2.00") - Decimal("5.00") = **-3.00**

### §B — PF=None edge case (ADR-0008 propagation)

Fixture A summary: `{..., pf:3.75}`
Fixture B summary: `{..., pf:None}` (e.g., B run had no losses)

Hand-computed deltas:
- pf: A=3.75 / B=None → delta column displays `"n/a"` per ADR-0008 (deviation from finance convention to None for both edge cases means delta is undefined).

### §C — Per-trade diff (4 signals, 3 differ)

Fixture: 4 signals (101, 102, 103, 104) with paper_trades for both runs:

| signal_id | A.close_reason | A.realized_pnl | B.close_reason | B.realized_pnl | Differs? |
|-----------|----------------|----------------|----------------|----------------|----------|
| 101       | sl             | -5.00          | tp             | 10.00          | YES (close_reason + pnl) |
| 102       | tp             | 10.00          | tp             | 15.00          | YES (pnl only) |
| 103       | tp             | 8.00           | tp             | 8.00           | NO (identical) |
| 104       | sl             | -3.00          | tp             | 20.00          | YES (close_reason + pnl) |

Expected output: 3 rows for 101/102/104; row count summary `"Per-trade differences (3 of 4 common signals diverged):"`.

### §D — Empty diff

Both runs have identical outcomes for all common signals → SQL returns 0 rows → `_format_per_trade_diff([])` returns `"No diverging trades found.\n"`.
