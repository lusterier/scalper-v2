## Algorithm specification

```python
class HistoricalSignalSource:
    """Async-iterable cursor-streamed historical signal replay.

    Yields SignalRow per BRIEF §12.2:1956. Ordered by received_at ASC.
    Filtered by symbol_universe + time window + ingestion_status='validated'.
    Server-side cursor for constant memory; T-502 ReplayBus owns pace.
    """

    def __init__(
        self,
        pool: asyncpg.Pool,
        *,
        bot_id: str,
        symbol_universe: list[str],
        from_at: datetime,
        to_at: datetime,
    ) -> None:
        if not symbol_universe:
            raise ValueError("HistoricalSignalSource: symbol_universe must not be empty")
        if to_at <= from_at:
            raise ValueError("HistoricalSignalSource: to_at must be > from_at")
        self._pool = pool
        self._bot_id = bot_id
        self._symbol_universe = list(symbol_universe)
        self._from_at = from_at
        self._to_at = to_at

    async def __aiter__(self) -> AsyncIterator[SignalRow]:
        async with self._pool.acquire() as conn:
            async with conn.transaction():
                async for row in conn.cursor(
                    _SELECT_SIGNALS_REPLAY_SQL,
                    self._symbol_universe,
                    self._from_at,
                    self._to_at,
                    prefetch=_DEFAULT_PREFETCH,
                ):
                    yield _row_to_signal_replay(row)
```

`_SELECT_SIGNALS_REPLAY_SQL`:
```sql
SELECT id, received_at, schema_version, source, idempotency_key,
       symbol, original_symbol, action, payload, ingestion_status, correlation_id
FROM signals
WHERE symbol = ANY($1)
  AND received_at >= $2
  AND received_at < $3
  AND ingestion_status = 'validated'
ORDER BY received_at ASC
```

`_DEFAULT_PREFETCH = 1000` (asyncpg default chunk size; tunable F5+ if needed; not exposed as constructor param — §N9-exempt infra tuning constant per L-001 active control, not a business knob).

**Cursor + transaction lifetime caveat**: `async for row in conn.cursor(...)` consumes within the wrapping `conn.transaction()` block — once iteration starts, the conn is held from first yield until iterator exhaustion or consumer error. Acceptable for T-507 CLI single-bot replay (one conn, no contention; pool-size of 1 sufficient). T-509 worker concurrency (running multiple backtests in parallel inside analytics-api lifespan) must size pool accordingly — flagged out-of-scope for T-504 but visible here so future T-509 plan-doc author sees the constraint.

`_row_to_signal_replay(row) -> SignalRow`:
- Mirrors `_row_to_signal` in analytics.py:706 if it exists (reuse). Otherwise inline-defined: extract fields, JSONB-decode `payload` defensively (`json.loads(row["payload"]) if isinstance(row["payload"], str) else row["payload"]`), narrow `action` via `Action(str(row["action"]))` enum, `ingestion_status` via `IngestionStatus(str(row["ingestion_status"]))`.

## Hand verification

### §A — Chronological order

Mock cursor yields 3 rows with received_at = `2026-04-01T12:00:00+00:00`, `2026-04-01T12:05:00+00:00`, `2026-04-01T13:00:00+00:00` (already in ASC order from SQL). Iterator yields all 3 in same order. Total 3 yields.

### §B — Symbol-universe filter

Constructor `symbol_universe=["BTCUSDT", "ETHUSDT"]`. SQL bind `$1 = ['BTCUSDT', 'ETHUSDT']`. Mock returns 2 rows (BTCUSDT + ETHUSDT). Test asserts `symbol = ANY($1)` filter applied (no SOLUSDT row even if mock returned it).

### §C — Time window inclusive-from / exclusive-to

Constructor `from_at=2026-04-01T00:00:00+00:00, to_at=2026-05-01T00:00:00+00:00`. SQL bind `$2 = from_at`, `$3 = to_at`. Half-open interval per existing analytics.py builder convention (`>= $2 AND < $3`).

### §D — `ingestion_status='validated'` only

SQL hardcodes `AND ingestion_status = 'validated'` per OQ-4=A. Test asserts duplicate / invalid signals NOT yielded (mock cursor yields only validated row even if test fixture includes others — SQL filter implicit).

### §E — Empty symbol_universe → ValueError

Constructor with `symbol_universe=[]` raises `ValueError`. Test pin per OQ-7=A defensive validation (system boundary — caller passes from BotConfig; defensive against config-loader bugs).

### §F — Empty result → empty iterator

Mock cursor yields zero rows; iterator completes after zero yields. No error.

### §G — JSONB payload-as-str decode (no-codec path)

Mock row `payload = '{"action": "LONG"}'` (str). Decoded via `json.loads` to dict `{"action": "LONG"}`. SignalRow.payload is dict. L-011 read-side robustness.

### §H — JSONB payload-as-dict passthrough (codec path)

Mock row `payload = {"action": "LONG"}` (dict, as analytics-api codec returns). Passes through unchanged. Same SignalRow.payload dict shape.
