## Decisions committed

Per the T-204 → T-205 → T-207 calibration trajectory.

1. **OQ-defaults applied throughout**; no unresolved decisions remain.

2. **`BybitV5Adapter` is per-instance state** (§N6): `_client`, `_limiter`, `_bus`, `_sub_account`, `_metrics_counter`, `_leverage_cache`. No module-level mutable state; module-level `logger` is read-only constant.

3. **`@idempotent` / `@non_idempotent` markers ON the public methods** per T-201 ExchangeClient Protocol baseline. T-208a public methods inherit Protocol-side markers; explicit decoration at impl site for T-206 contract framework cross-check.

4. **Method-level rate-limit + metrics decorator pattern**: each write method follows the same envelope:
   ```python
   await self._limiter.acquire(self._sub_account, "<group>")
   try:
       result = await self._client.request(...)
   except RateLimitError:
       if self._metrics_counter is not None:
           self._metrics_counter.labels(exchange="bybit", endpoint_group="<group>").inc()
       await self._limiter.signal_upstream_rate_limit()
       raise
   ```
   `place_market_order` adds an extra catch for `NetworkTimeout` → `UnknownState(last_known_action="place_market_order")` per §11.3 H-003 binding.

5. **Retry counts per method per brief §11.2**:
   - `set_leverage` → 3 (idempotent)
   - `place_market_order` → 0 (H-003 zero-retry)
   - `set_trading_stop` → 3 (idempotent)
   - `cancel_order` → 3 (idempotent)

6. **LRU cache (set_leverage) is per-instance**: stdlib `functools.lru_cache` doesn't support `async` directly; instead use a `dict[tuple[str, int], datetime]` with TTL check. Cache hit only if `now - cached_at < self._leverage_cache_ttl_s` (per L-001 §N9 fix — operator-tunable via ctor kwarg, default 3600.0s per Q9 brief). Eviction on miss-after-expiry. Cache size bounded by symbol count (≤ symbols-per-bot ≤ 10 per F2 spec).

7. **Bybit V5 wire-payload snapshots verbatim per docs** (Decision binds at impl site):
   - `set_leverage` POST `/v5/position/set-leverage` body `{"category":"linear", "symbol": symbol, "buyLeverage": str(leverage), "sellLeverage": str(leverage)}`.
   - `place_market_order` POST `/v5/order/create` body `{"category":"linear", "symbol": symbol, "side": "Buy"|"Sell", "orderType": "Market", "qty": str(qty), "reduceOnly": reduce_only}`.
   - `set_trading_stop` POST `/v5/position/trading-stop` body `{"category":"linear", "symbol": symbol, "stopLoss": str(sl_price)?, "takeProfit": str(tp_price)?, "tpSize": str(tp_size)?, "tpslMode": tpsl_mode}` — empty/None fields omitted from body.
   - `cancel_order` POST `/v5/order/cancel` body `{"category":"linear", "symbol": symbol, "orderId": order_id}`.

8. **Side mapping single function**: `_to_bybit_side(side: Literal["buy","sell"]) -> Literal["Buy","Sell"]` — small helper. No new public surface.

9. **`OrderPlaceResult` construction from response**: Bybit V5 response `result` field has `{orderId, orderLinkId}`; map to `OrderPlaceResult(exchange_order_id=str(result["orderId"]), placed_at=now_fn())`. `placed_at` is local timestamp (Bybit response timestamp is in `time` envelope field but ms-precision unreliable for our purposes; local UTC sufficient per §N1).

### `packages/exchange/bybit_v5/adapter.py` (NEW)

```python
"""§11.2 BybitV5Adapter — ExchangeClient Protocol implementation (T-208a).

T-208a ships 4 write methods + ctor:
* :meth:`set_leverage` — LRU-cached 1h TTL per Q9 default.
* :meth:`place_market_order` — H-003 zero-retry; NetworkTimeout → UnknownState.
* :meth:`set_trading_stop` — H-013 no-default tpsl_mode (T-201 contract).
* :meth:`cancel_order` — 3× retry per §11.2.

Stub forward-pointers per T-211 stub-pin precedent:
* ``stream_executions``/``stream_positions``/``close``: T-209.
* ``get_positions``/``get_fill_price``/``get_closed_pnl_cumulative``: T-208b.

Composes T-207 BybitV5Client (HTTP envelope + signing + retry matrix)
+ T-205 SharedRateLimiter (cross-bot IP coordination per H-025) +
optional Prometheus Counter (rate_limit_hits_total per §15.3).
"""

from __future__ import annotations

import logging
from datetime import datetime, timedelta
from typing import TYPE_CHECKING, Any, Literal

from packages.core import idempotent, non_idempotent, now_utc
from packages.exchange.errors import NetworkTimeout, RateLimitError, UnknownState
from packages.exchange.types import (
    ExecutionEvent,
    OrderPlaceResult,
    Position,
    PositionEvent,
)

if TYPE_CHECKING:
    from collections.abc import AsyncIterator, Callable
    from decimal import Decimal

    from prometheus_client import Counter

    from packages.bus import NatsClient
    from packages.exchange.bybit_v5.client import BybitV5Client
    from packages.exchange.rate_limiter import SharedRateLimiter


__all__ = ["BybitV5Adapter"]

_CATEGORY = "linear"
_DEFAULT_LEVERAGE_CACHE_TTL_S = 3600.0  # Q9 brief default — 1h TTL; operator-tunable via ctor kwarg per L-001 §N9.
_BUY: Literal["Buy"] = "Buy"
_SELL: Literal["Sell"] = "Sell"

logger = logging.getLogger(__name__)


def _to_bybit_side(side: Literal["buy", "sell"]) -> Literal["Buy", "Sell"]:
    return _BUY if side == "buy" else _SELL


def _stub_message(method: str, owner: str) -> str:
    return f"BybitV5Adapter.{method} body lands at {owner}"


class BybitV5Adapter:
    """ExchangeClient implementation against Bybit V5 REST + private WS."""

    def __init__(
        self,
        *,
        client: BybitV5Client,
        limiter: SharedRateLimiter,
        bus: NatsClient,
        sub_account: str,
        metrics_counter: Counter | None = None,
        leverage_cache_ttl_s: float = _DEFAULT_LEVERAGE_CACHE_TTL_S,
        now_fn: Callable[[], datetime] = now_utc,
    ) -> None:
        self._client = client
        self._limiter = limiter
        self._bus = bus
        self._sub_account = sub_account
        self._metrics_counter = metrics_counter
        self._leverage_cache_ttl_s = leverage_cache_ttl_s
        self._now_fn = now_fn
        # Per-instance leverage cache: key=(symbol, leverage), value=cached_at.
        self._leverage_cache: dict[tuple[str, int], datetime] = {}

    @idempotent
    async def set_leverage(self, symbol: str, leverage: int) -> None:
        """LRU-cached per Q9 default; on cache hit return None silently."""
        key = (symbol, leverage)
        cached_at = self._leverage_cache.get(key)
        if cached_at is not None and (self._now_fn() - cached_at) < timedelta(
            seconds=self._leverage_cache_ttl_s,
        ):
            return
        await self._limiter_acquire("positions")
        try:
            await self._client.request(
                "POST", "/v5/position/set-leverage",
                body={
                    "category": _CATEGORY,
                    "symbol": symbol,
                    "buyLeverage": str(leverage),
                    "sellLeverage": str(leverage),
                },
                retries=3,
            )
        except RateLimitError:
            self._on_rate_limit_hit("positions")
            raise
        self._leverage_cache[key] = self._now_fn()

    @non_idempotent
    async def place_market_order(
        self, symbol: str, side: Literal["buy", "sell"], qty: Decimal,
        reduce_only: bool = False,
    ) -> OrderPlaceResult:
        """H-003: 0 retries; NetworkTimeout → UnknownState."""
        await self._limiter_acquire("orders")
        try:
            result = await self._client.request(
                "POST", "/v5/order/create",
                body={
                    "category": _CATEGORY,
                    "symbol": symbol,
                    "side": _to_bybit_side(side),
                    "orderType": "Market",
                    "qty": str(qty),
                    "reduceOnly": reduce_only,
                },
                retries=0,
            )
        except RateLimitError:
            self._on_rate_limit_hit("orders")
            raise
        except NetworkTimeout as exc:
            raise UnknownState("place_market_order") from exc
        return OrderPlaceResult(
            exchange_order_id=str(result["orderId"]),
            placed_at=self._now_fn(),
        )

    @idempotent
    async def set_trading_stop(
        self, symbol: str, tpsl_mode: Literal["Full", "Partial"],
        sl_price: Decimal | None = None,
        tp_price: Decimal | None = None,
        tp_size: Decimal | None = None,
    ) -> None:
        """H-013: tpsl_mode no-default (T-201 Protocol contract)."""
        body: dict[str, Any] = {
            "category": _CATEGORY,
            "symbol": symbol,
            "tpslMode": tpsl_mode,
        }
        if sl_price is not None:
            body["stopLoss"] = str(sl_price)
        if tp_price is not None:
            body["takeProfit"] = str(tp_price)
        if tp_size is not None:
            body["tpSize"] = str(tp_size)
        await self._limiter_acquire("positions")
        try:
            await self._client.request(
                "POST", "/v5/position/trading-stop", body=body, retries=3,
            )
        except RateLimitError:
            self._on_rate_limit_hit("positions")
            raise

    @idempotent
    async def cancel_order(self, symbol: str, order_id: str) -> None:
        await self._limiter_acquire("orders")
        try:
            await self._client.request(
                "POST", "/v5/order/cancel",
                body={"category": _CATEGORY, "symbol": symbol, "orderId": order_id},
                retries=3,
            )
        except RateLimitError:
            self._on_rate_limit_hit("orders")
            raise

    # T-208b stubs ----------------------------------------------------------

    @idempotent
    async def get_positions(self, symbol: str | None = None) -> list[Position]:
        raise NotImplementedError(_stub_message("get_positions", "T-208b"))

    @idempotent
    async def get_fill_price(self, symbol: str, order_id: str) -> Decimal | None:
        raise NotImplementedError(_stub_message("get_fill_price", "T-208b"))

    @idempotent
    async def get_closed_pnl_cumulative(self, sub_account: str) -> Decimal:
        raise NotImplementedError(_stub_message("get_closed_pnl_cumulative", "T-208b"))

    # T-209 stubs -----------------------------------------------------------

    def stream_executions(self) -> AsyncIterator[ExecutionEvent]:
        raise NotImplementedError(_stub_message("stream_executions", "T-209"))

    def stream_positions(self) -> AsyncIterator[PositionEvent]:
        raise NotImplementedError(_stub_message("stream_positions", "T-209"))

    async def close(self) -> None:
        raise NotImplementedError(_stub_message("close", "T-209"))

    # Internals -------------------------------------------------------------

    async def _limiter_acquire(self, group: Literal["orders", "positions"]) -> None:
        await self._limiter.acquire(self._sub_account, group)

    def _on_rate_limit_hit(self, group: str) -> None:
        if self._metrics_counter is not None:
            self._metrics_counter.labels(exchange="bybit", endpoint_group=group).inc()
        # signal_upstream_rate_limit is async — we await it from the caller.
```

## Lifecycle

`BybitV5Adapter` instance lifecycle (T-208a scope):

1. Constructor: store DI'd dependencies; init empty `_leverage_cache`. NO I/O.
2. `set_leverage(symbol, leverage)`: cache hit → return None; else acquire limiter → call upstream → update cache.
3. `place_market_order(symbol, side, qty, reduce_only)`: acquire limiter → call upstream (retries=0) → catch NetworkTimeout → UnknownState; catch RateLimitError → metrics + signal upstream + re-raise; on success → return OrderPlaceResult.
4. `set_trading_stop(symbol, tpsl_mode, sl_price?, tp_price?, tp_size?)`: build body with optional fields → acquire limiter → call upstream (retries=3) → catch RateLimitError → metrics + signal + re-raise.
5. `cancel_order(symbol, order_id)`: acquire limiter → call upstream → catch RateLimitError → metrics + signal + re-raise.
6. Stubbed methods (T-208b/T-209): raise NotImplementedError immediately.

T-208a does NOT implement `close()` — T-209 owns lifecycle.

## Edge cases

1. **Concurrent `set_leverage` calls for same symbol within TTL**: race between two coroutines — both check cache (miss), both call upstream. Upstream is idempotent (Bybit accepts duplicate set), so safe. Cache updated by whichever returns first; second overwrites with later timestamp. Acceptable per Q9 default.

2. **`place_market_order` returns 200 OK with retCode==0 but missing `orderId`**: `result["orderId"]` raises `KeyError` → propagates as untyped exception. **Edge-case-test-pin** asserts this loud-failure (Bybit guarantees orderId on success per docs).

3. **`set_trading_stop` with all None pricing fields**: body has only `category`/`symbol`/`tpslMode`. Bybit will reject with retCode 10001 → `OrderRejected(reason)` propagates. T-208a tests pin "tpsl_mode without prices" path.

4. **`cancel_order` with order_id of already-cancelled order**: Bybit returns specific retCode (typically 110001 or similar "order not found"). T-208a doesn't catch — propagates as `ExchangeError`. Caller (T-218 dispatcher) treats as benign per §11.2 `cancel_order @idempotent`.

5. **`leverage=0`**: Bybit rejects. T-208a doesn't validate; upstream returns retCode → `OrderRejected`. Acceptable per §0.8 anti-hypothetical.

6. **Cache evicted under memory pressure**: not a concern — bounded by symbol count ≤10. No LRU eviction implemented (TTL-only).

7. **`now_fn` returns timezone-naive datetime**: `now_utc` returns aware UTC; if test injects naive, `(naive - aware) < timedelta(...)` raises TypeError. Acceptable — caller responsibility to inject UTC-aware now_fn.
