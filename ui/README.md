# scalper-v2 UI

Dashboard frontend per BRIEF §14. React 18 + Vite + TypeScript (strict) + Tailwind CSS + shadcn/ui + TanStack Router/Query + Zustand + Recharts.

T-410 ships the scaffold only; subsequent UI tasks (T-411 component library, T-412..T-420 nine sections) land on top.

## Prerequisites

- **Node.js 20.19+** (host has 20.19.6 via nvm; required by `@tanstack/router-plugin@1.167+` which depends on `unplugin@3` requiring `import.meta.dirname` from Node 20.11+). Operator-led bump 2026-05-04 from Node 18.19.1.
- **pnpm 10.33.2** — preferred install via nvm-shimmed npm:
  ```bash
  source ~/.nvm/nvm.sh
  nvm use 20.19
  npm install -g pnpm@10.33.2
  ```
  Fallback for hosts without nvm — user-prefix npm install:
  ```bash
  mkdir -p ~/.npm-global && npm config set prefix ~/.npm-global
  npm install -g pnpm@10.33.2
  echo 'export PATH="$HOME/.npm-global/bin:$PATH"' >> ~/.bashrc
  source ~/.bashrc
  ```
- **analytics-api running locally** on `http://127.0.0.1:8000` BEFORE `pnpm dev`. The Vite dev server proxies `/api/*` and `/events/*` to the backend. Without it, every fetch fails with `ECONNREFUSED`.

  Start the backend (separate terminal, repo root):
  ```bash
  uv run uvicorn services.analytics_api.app.main:create_app --factory --host 127.0.0.1 --port 8000
  ```

## First-time setup

```bash
cd ui
pnpm install --frozen-lockfile  # deterministic install per pnpm-lock.yaml
```

Cold install ~30-60s; produces ~1-2 GB `node_modules/`.

## Development

```bash
pnpm dev          # Vite dev server on http://127.0.0.1:5173
pnpm typecheck    # TypeScript strict mode check (no emit)
pnpm lint         # ESLint flat config
pnpm test         # Vitest watch mode
pnpm test:ci      # Vitest single run (CI-friendly)
pnpm build        # Production bundle (sanity check; not deployed in F4)
pnpm preview      # Serve `dist/` locally for build verification
pnpm format       # Prettier with Tailwind class ordering
```

## Layout

```
ui/src/
├── main.tsx              # Entry: ReactDOM + RouterProvider + QueryClientProvider
├── routeTree.gen.ts      # Generated by @tanstack/router-plugin (committed)
├── styles/
│   ├── theme.css         # CSS variables (BRIEF §14.5)
│   └── globals.css       # Tailwind directives + theme.css import
├── routes/               # File-based routes
│   ├── __root.tsx        # Layout shell (left nav + content)
│   └── index.tsx         # `/` placeholder
├── components/ui/        # shadcn/ui copy-paste (per BRIEF §14.1:2046)
├── lib/
│   ├── utils.ts          # cn() helper
│   ├── api-client.ts     # Native fetch wrapper
│   └── sse-client.ts     # Native EventSource wrapper
└── __tests__/            # Vitest smoke tests
```
